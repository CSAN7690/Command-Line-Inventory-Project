# Funko Pops Inventory System

## Setup

1. Clone the repository
2. Run `npm install`

## Usage 

### ➕ Add a new item

```node index.js add```

### 📋 List all items
```node index.js list```

### 🔎 View details of an item
````node index.js view [id]```

### 🗑️ Delete an item 
```node index.js delete [id]```

### ⚒️ Update an item
```node index.js update [id]```

### ✚ 🛒 Add to cart
```node index.js cart add [id]```

### 🚫 🛒 Cancel cart
``` node index.js cart cancel```
